# Лабораторна робота №6 (на основі ЛР5) — Vite + SCSS

## Запуск
```bash
npm install
npm run dev
```

## Структура SCSS
- `src/scss/main.scss` — головний файл, який імпортує partial-и через `@use`
- `src/scss/_*.scss` — partial-файли зі стилями (з ЛР5), розбиті за секціями

## Примітка по зображеннях
Зображення розміщені у `public/img`, тому в HTML використовується шлях виду `/img/...`.
